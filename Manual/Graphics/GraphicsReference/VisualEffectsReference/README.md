Visual Effects Reference

视觉效果可以应用于相机，游戏对象，光源和游戏的其他元素。本节提供在Unity编辑器中可用视觉效果的信息。

要访问Unity编辑器中可用的视觉效果，请在 Hierarchy 中选择要应用效果的对象，然后在  Inspector 中转到 **Add Component > Effects**。
